#  Glassmorphism UI Examples

These examples demonstrate how to use glassmorphism styles in real UI components.

---

##  Glass Card

```html
<div class="glass-card">
  <h2>Glass Card</h2>
  <p>This is a frosted glass effect card.</p>
</div>
```

---

##  Glass Navbar

```html
<nav class="glass-navbar">
  <h1>Logo</h1>
  <ul>
    <li>Home</li>
    <li>About</li>
    <li>Contact</li>
  </ul>
</nav>
```

---

##  Glass Button

```html
<button class="glass-button">Click Me</button>
```

---

##  Glass Modal

```html
<div class="glass-modal">
  <h3>Modal Title</h3>
  <p>This is a glassmorphism modal.</p>
</div>
```
